# Harness Workspace Starter

这是 Harness Engineering 课程的练习项目。

## 项目说明

这是一个简化的文档管理应用骨架，用于练习 harness 设计。

## 技术栈

- TypeScript
- Node.js
- Vitest (测试)

## 命令

```bash
# 安装依赖
npm install

# 类型检查
npm run type-check

# 运行测试
npm test

# 开发模式
npm run dev
```

## 练习用功能

- feat-001: 文档导入 (TXT/PDF)
- feat-002: 文档列表展示
- feat-003: AI 问答
- feat-004: 索引与持久化
